# PhysiqueOS Roadmap

This roadmap represents the planned evolution of PhysiqueOS from prototype to production.

The roadmap is organized by milestones rather than release dates.

---

# Founder Alpha Success Metric

Founder Alpha is successful when the founder instinctively opens PhysiqueOS instead of messaging ChatGPT after entering new evidence because the app provides the same level of clarity, confidence, and actionable guidance.

# Future Milestone

## Conversational Intelligence 

Future milestones include:

* Conversational onboarding
* Conversational logging
* Speech-to-Evidence engine
* Richer entity resolution
* Additional specialized voice interpreters
* User correction learning
* Voice-driven historical backfill
* Voice-driven protocol updates

Long-term, voice should become the lowest-friction way to teach PhysiqueOS about the user and continuously submit new evidence.

The objective is not speech-to-text.

The objective is speech-to-structured-evidence.

Users should rarely feel like they are filling out forms.

Foundational Voice Intelligence architecture is defined in `docs/VOICE_INTELLIGENCE.md`.

**Status:** Foundation complete; product maturity planned

# Guiding Principles

Every feature should satisfy at least one of these objectives:

* Reduce uncertainty
* Improve confidence
* Improve prediction quality
* Increase adherence
* Reduce cognitive load
* Help users achieve meaningful health goals

If a feature does not support one of these principles, it should be reconsidered.
